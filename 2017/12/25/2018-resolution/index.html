2018--宁愿傲娇，也勿泄气

发表于 2017-12-25 | 分类于 生活与创作 |

2018 年，学会把重要的事情放在第一位，少点其他事情的分心。打好基础在前，追新技术在后，以不变应万变。能够“很有效率”的实现工作中的需求，不加班及有时间做其他有趣的事情。

一、前端&编码

  • 能够很熟练的写 HTML+ CSS ,重复的页面，相同的样式，能够提取和重构。
  • 切实打好 JavaScript 基础，能够快速原生手写常用的组件。
  • 开发自适应和响应式的网站成为标配
  • 至少熟悉一种打包工具，能够自行根据项目配置开发环境。
  • 了解 ES6 的所有新特性
  • 能熟练使用一种前端框架，如 vue ng 和 react（vue → react）
  • 理解前端常用设计模式，如闭包、面向对象、MVC、MVVM、模块化、组件化。
  • 熟悉 HTTP，了解请求 响应 缓存 代理 反向代理 https 等
  • 在 GitHub 至少有一个 star 超过 10 的开源代码作品
  • 至少写一篇超过 10k+阅读量的博客。

二、设计

不编码时，可以搞搞设计嘛，满足自己就行 😄。跟着学习一些国内主流网站潮流走啦~

  • 一款音乐类网站设计（手机端 + PC 端）
  • 一款运动类网站设计（喜欢跑步啦、篮球啦、台球也不错）
  • 一款 VR 网站（到底是什么的。。。待研究）

三、 运动

  • 参加一次半马（坚持每周跑三次步以上）
  • 倒挂引体向上

四、读书

有位大佬推荐《如何阅读一本书》，把里面的方法时间，它把阅读分为“基础阅读”、“检视阅读”、“分析阅读”、“主体阅读”，希望自己能够学会阅读，达到 “检视阅读”之上，开始“分析阅读”的层次。

编码计算机类
  • 《编码》
  • 《穿越计算机的迷雾》
  • 《计算机系统概论 : （原书第 2 版）》
  • 《Computer Systems》《深入理解计算机系统（原书第 2 版）》
  • 《Joel 谈优秀软件开发方法》
个人成长类
  • 《富爸爸，穷爸爸》系列
  • 《远大前程》
  • 《软件开发者路线图》
  • 《Apprenticeship Patterns》《软件开发者路线图》
  • 《影响力》
小说类
  • 《新名字的故事》
  • 《围城》
  • 东野先生的《悖论》、《湖畔杀入案》

五、生活

  • 献一次血
  • 扇贝英语坚持打卡 200 天以上
  • 拿到英语六级证书
  • 完成《有些事情现在不做了，一辈子都不会做了》的十件事以上。
  • 给自己拍一套个人写真

六、旅行

  • 坐一次火车去旅行（就算一个人，也要去啦~,看过高木直子《一个人去旅行》之感）

七、摄影

  • 拍人像时，减少自己的拍照量，提高自己的出片率，从一天约拍一次拍 400 多张精简至 200 多张。
  • 提升后期的修片速度，少于一周内可以出片。

（完）
